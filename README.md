Organizador de Vendas Multi-Plataforma

Projeto de organização de vendas de varias plataformas em resumo.
Feito para facilitar a visão de quantidade, tipo e valor dos produtos vendidos em todas plataformas.
O projeto foi criado com foco em aprendizado prático de back-end e manipulação de dados.

FUNCIONALIDADES
-Upload e leitura de planilhas de vendas .xlsx
-consolidação de dados em resumo
-armazenamento de informações em banco de dados SQLITE3
-Interface web simples para vizualização de dados



Tecnologia utilizadas
-Python
-flask
-Sqlite
-Html-CSS
-Pandas

para executar projeto clone o repositorio.
utilizando start_flask, abre direto em site e terminal
entre no projeto e installe flask pandas / 
executar - python app.py
para acesso no navegador http://localhost:5000


IMPORTANTE: O projeto só lê arquivos .xlsx extraídos diretamente das plataformas.
Os arquivos reais de vendas não estão no repositório por motivos de privacidade. Use seus próprios arquivos ou exemplos fictícios para testar.
